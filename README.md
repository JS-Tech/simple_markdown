# SimpleMarkdown

This project rocks and uses MIT-LICENSE.

# Features

* Titles with
```
# Title 1
## Title 2
...
###### Title 6
```
* List with (only)
```
* Apple
* Banana
```
* Emphasis with
```
This is *emphasis* test
```
* Bold with
```
This is **bold test**
```
* Image with
```
This is an image : [description](http://example.com/im.png)
```
* Link with
```
This is a ![link](http://example.com)
```
